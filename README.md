# map-info

## Project setup
```
npm git clone git@github.com:romkevi4/map-info.git
```

```
npm install
```

### Start a dummy json-server
```
json-server --watch db.json
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
